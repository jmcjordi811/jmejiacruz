The third notation is called Big Omega Notation.

Big Ώ Notation is used to identify asymptotic lower bounds.

Definition A function t(n) [running time] is said to be in θ(g(n)), t(n) is an element in θ(g(n)), if t(n) is bounded below by some positive constant multiple of g(n) for all large n.
