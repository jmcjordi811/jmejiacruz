The second notation is called Big O Notation.

Big O Notation is used to identify asymptotic upper bounds.

Definition A function t(n) [running time] is said to be in O(g(n)), t(n) is an element in O(g(n)), if t(n) is bounded above by some constant multiple of g(n) for all large n.
